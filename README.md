# Code Challenge

## Features:

- [Typescript](https://www.typescriptlang.org/) 4.9.5
- [React](https://react.dev/) 18.2.0
- [Node](https://nodejs.org/en/) 18

## Developer setup

1. Install Node
2. Clone the repository
3. Install dependencies

```bash
npm ci
```

4. Start the application

```bash
npm run start
```
